# [규현팀] 당근마켓 클론코딩

## 🥕 당근마켓 

> 당근 마켓 중고거래 서비스를 클론 코딩한 프로젝트 입니다.

## 🥕 프로젝트의 목적(Purpose)

> 핵심 도메인을 추출하고 주요 기능을 분석 및 구현하며 협업을 경험해 봅니다. </br>
> JIRA를 사용하여 스크럼 등 실무에서 사용하는 개발 방법론을 통해 협업 역량을 강화해 봅니다.
> 
> 프로젝트 기간 : 2023/08/26 ~ 2023/09/22
